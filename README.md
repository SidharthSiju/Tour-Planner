    -   Framework: [Next.JS](https://nextjs.org/) (a React Framework)

    -   Package Manager: [Yarn](https://yarnpkg.com/)

## How to Run Locally

1. Install [yarn](https://yarnpkg.com/getting-started/install)
2. Project Setup
    ```shell
    yarn install
    ```
3. Run the Project in Development Mode
    ```shell
    yarn dev
    ```
